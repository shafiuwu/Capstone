# 🚀 Impulsa – Plataforma de Voluntariado

Impulsa es una plataforma web que conecta voluntarios con organizaciones, facilitando la búsqueda y gestión de oportunidades de voluntariado de forma simple y personalizada.

---

## 📌 Descripción

El objetivo del proyecto es reducir la brecha entre personas que desean ayudar y organizaciones que necesitan apoyo.

La plataforma permite a los usuarios registrarse, explorar oportunidades y recibir recomendaciones según sus intereses, mientras que las organizaciones pueden publicar y gestionar voluntariados.

---

## ⚙️ Tecnologías

**Backend**
- Node.JS

**Base de Datos**
- PostgreSQL

**Arquitectura**
- API REST

**Otros**
- Chatbot con Inteligencia Artificial
- Sistema de recomendaciones (Machine Learning)

**Frontend**
- React

---

## 🧩 Funcionalidades

- Registro y autenticación de usuarios  
- Creación de perfiles  
- Publicación de oportunidades  
- Postulación a voluntariados  
- Recomendaciones personalizadas  
- Notificaciones por correo  
- Chatbot de asistencia
