🚀 Impulsa – Plataforma de Voluntariado

Impulsa es una plataforma web que conecta voluntarios con organizaciones, facilitando la búsqueda, gestión y recomendación de oportunidades de voluntariado de forma simple y personalizada.

📌 Descripción

El proyecto nace con el objetivo de reducir la brecha entre personas que desean ayudar y organizaciones que necesitan apoyo.

Impulsa permite a los usuarios registrarse, explorar oportunidades y recibir recomendaciones basadas en sus intereses, mientras que las organizaciones pueden publicar y gestionar sus necesidades de voluntariado.

Backend

Node.JS

Frontend

React

Base de Datos

PostgreSQL

Arquitectura

API REST

Otras herramientas

Chatbot con Inteligencia Artificial
Sistema de recomendaciones (Machine Learning)

🧩 Funcionalidades principales
Registro y autenticación de usuarios
Creación de perfiles de voluntarios
Publicación de oportunidades por organizaciones
Postulación a voluntariados
Sistema de recomendaciones personalizadas
Notificaciones por correo electrónico
Chatbot de asistencia con IA
🎯 Objetivo

Facilitar la conexión entre voluntarios y organizaciones, promoviendo la participación comunitaria y generando impacto social positivo.
